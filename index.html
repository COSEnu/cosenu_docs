COSEν

A SOPHISTICATED ENGINE FOR COLLECTIVE NEUTRINO OSCILLATION SIMULATION

About

COSEν provide different numerical simulation methods, the finite difference and the finite volume scheme to simulate fast pairwise collective neutrino conversions. The finite differencing scheme uses fourth order accurate central differencing scheme with third order Kreiss-Oliger dissipation to treat the advection effectively. The finite volume scheme uses seventh order accurate weighted non-oscillatory scheme for the flux reconstruction across the grid boundaries. In both cases time evolution is implemented using fourth order Runge-kutta method so that the spatial and temopral accuracies are of the same order.
